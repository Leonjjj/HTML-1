# HTML入门笔记1
## HTML是谁发明的
* HTML是李爵士发明的
## HTML起手应该写什么
```
<!DOCTYPE html>

<html>

<head>

<title></title>

</head>

<body></body>

</html>
```
## 常用表示章节的标签
* H1~H6：按顺序表示六种从大到小的标题
* section:表示章节
* article:表示文章
* P：表示段落
* header:表示头部
* footer:表示脚步
* main：表示主要内容
* aside:表示旁支内容
* div：划分

## 全局属性有哪些
* class
* contenteditable：可编辑
* hidden：隐藏
* id
* style
* tabindex：赋予tab功能 0最后 -1屏蔽
* title：标题
## 常用内容标签
* ol+li：有序排列
* ul+li：无序
* dl+dt+did 描述列表，其中dt代表的是描述对象，dd代表的是描述内容
* per:显示多个空格和换行
* hr：水平分割线
* br：换行
* a：给链接
* em：语气上强调
* strong：内容上强调
* code：字体等宽
* q：quote缩写内联引用
* blockquote：换行引用
